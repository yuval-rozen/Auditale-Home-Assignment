Frequently asked questions

How is the health score calculated?

We combine engagement recency, feature coverage, ticket volume, and invoice timeliness with weights we can tune on your demand. Missing data is treated neutrally.

Does it update in real time?

Yes — scores compute on request from the latest tables, so the dashboard reflects current signals.

Can I see why a customer’s score changed?

Yes. In the detail view, each factor is displayed separately. For example, if logins dropped in the last 30 days, you’ll see the Login Score decrease, which lowers the overall Health Score.

How do I find a specific customer in the dashboard?

Use the search bar at the top of the dashboard. Type the customer’s name (or part of it), and the table will filter automatically to show only matching customers.
